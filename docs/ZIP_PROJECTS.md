# Zip 项目检查与解压记录

## 解压策略

对根目录 zip 进行去重检查后，非重复项目解压到 `projects/from-zips/`。解压时只保留关键内容：

- 保留：源码、配置、README、需求文件、数据集 `data.yaml`、小型文档和少量必要结果图。
- 跳过：虚拟环境、缓存、训练输出、权重、视频、大规模图片数据、标签文件、压缩包内压缩包。
- 二次裁剪：对 YOLO/Ultralytics/PaddleGAN 等上游框架副本，只保留项目入口脚本、配置和少量示例结果；完整框架与原始交付包保留在本地原目录或 zip 中。
