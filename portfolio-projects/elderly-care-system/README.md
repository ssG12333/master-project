# 失能老年人照护服务系统

## 项目简介

本项目是一个面向失能老年人照护场景的服务管理系统，围绕用户、健康档案、照护服务、服务订单和推荐内容建立业务流程。项目适合作为 Java 后端业务建模、CRUD 接口开发和基础前端页面整合的展示案例。

原始目录：`老年人项目全包/失能老年人/elderly-care-system/`

## 技术栈

- 后端：Spring Boot 4, Java 21, Spring Web MVC
- 数据访问：MyBatis
- 数据库：MySQL
- 前端：静态 HTML, Vue, Element UI, Axios, ECharts
- 工具：Maven, Lombok

## 主要功能

- 用户注册、登录、信息修改、用户列表和删除。
- 健康档案新增与列表查询。
- 照护服务项目的新增、查询、修改、删除。
- 服务订单创建、我的订单、全部订单和状态更新。
- 推荐内容列表查询。

## 工作链路

1. 静态前端页面提交用户、健康档案和服务订单请求。
2. Spring Boot Controller 按业务模块拆分接口。
3. Service 层封装用户、健康档案、服务项目、订单和推荐逻辑。
4. MyBatis Mapper 访问 MySQL 表数据。
5. 前端通过 Axios 获取列表、提交表单并刷新页面状态。

## 知识点

- Spring Boot 分层结构：Controller、Service、Mapper、Entity。
- MyBatis 映射和下划线转驼峰配置。
- REST 风格接口组织。
- MySQL 数据源配置。
- 静态前端页面与后端接口对接。
