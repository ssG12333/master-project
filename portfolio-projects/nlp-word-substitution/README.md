# 自然语言处理词义替换工具

## 项目简介

本项目实现英文文本中的词义替换流程，结合 spaCy、NLTK WordNet 和词形还原/变形工具，根据词性和语义候选生成替换词，并提供测试脚本验证替换结果。

原始目录：`自然语言处理词义替换/`

## 技术栈

- Python
- spaCy
- NLTK / WordNet
- pyinflect
- pandas
- scikit-learn

## 主要功能

- 读取训练数据和测试样例。
- 使用 spaCy 进行分词、词性标注和依存信息分析。
- 使用 WordNet 查询同义词和候选词。
- 根据原词词性进行词形匹配和变形。
- 提供测试框架检查替换结果。

## 工作链路

1. 读取文本数据和待替换词。
2. 使用 spaCy 分词、词性标注和语法分析。
3. 使用 WordNet 查询同义词和语义候选。
4. 通过 pyinflect 保持替换词的词形一致。
5. 运行测试脚本验证替换质量和边界情况。
