# BiLSTM 情感分析模型

## 项目简介

本项目基于 PyTorch 实现文本情感二分类，使用 NLTK 分词、词表构建、序列填充和 BiLSTM 分类模型完成评论情感预测。项目适合展示 NLP 文本分类、深度学习训练流程和实验参数对比能力。

原始目录：`情感分析/`

## 技术栈

- Python, PyTorch
- NLTK
- pandas, NumPy
- BiLSTM, Embedding
- Dataset/DataLoader
- matplotlib, tqdm

## 主要功能

- 从 CSV 读取评论文本和情感标签。
- 使用 NLTK 分词并构建高频词表。
- 将文本转换为定长 token 序列。
- 使用 Embedding + BiLSTM + 全连接层完成情感分类。
- 支持学习率、层数、批大小等实验参数配置。
- 记录训练指标并绘制曲线。

## 工作链路

1. 读取评论数据并完成文本清洗和标签解析。
2. 基于训练文本统计词频，构建有限大小词表。
3. 将句子分词、截断或补齐为固定长度序列。
4. 使用 BiLSTM 提取上下文特征并输出二分类概率。
5. 训练过程中记录损失、准确率和验证集表现。
6. 根据实验参数保存模型和训练曲线。

## 知识点

- 文本分词、词表构建和序列编码。
- Embedding 与 BiLSTM 文本表示。
- PyTorch 自定义 Dataset。
- 训练/验证流程和学习率调度。
- NLP 分类任务的实验组织。
