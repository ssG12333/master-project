# 股票预测与回测分析

## 项目简介

本项目面向股票价格和收益预测，包含技术指标计算、数据预处理、深度学习建模、滚动训练和回测分析。项目可展示金融时间序列特征工程、模型训练和策略评估流程。

原始目录：`股票预测/`

## 技术栈

- Python
- PyTorch, GRU
- pandas, NumPy
- scikit-learn
- statsmodels
- matplotlib, seaborn
- TA-Lib

## 主要功能

- 批量读取股票 CSV 数据。
- 计算技术指标并进行滚动归一化。
- 构建 GRU 和双分支 GRU 模型。
- 滚动窗口训练预测。
- 基于预测结果进行收益回测。
- 输出图表和指标分析。

## 工作链路

1. 批量读取股票历史行情和财务指标。
2. 计算技术指标并做滚动归一化。
3. 构造时间窗口样本，训练 GRU 或双分支 GRU。
4. 用滚动窗口方式预测未来走势，减少时间泄漏。
5. 将预测结果接入回测逻辑，输出收益和可视化分析。

## 知识点

- 金融时间序列特征工程。
- 循环神经网络建模。
- 滚动训练与时间泄漏控制。
- 回测收益计算。
- 预测结果可视化。
