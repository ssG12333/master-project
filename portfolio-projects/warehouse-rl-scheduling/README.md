# 智能仓储与多智能体任务分配

## 项目简介

本项目围绕智能仓储场景，研究仓储环境中的路径规划、任务分配和多智能体协同。代码中包含仓库环境建模、强化学习训练、对比算法实验、结果可视化，以及基于 CBBA 的多智能体任务分配实现。

原始目录：`智能仓储/`, `多智能体任务动态分配/`, `agv智能仓储/`

## 技术栈

- Python
- PyTorch
- CBBA 任务分配算法
- Matplotlib, Seaborn, Plotly
- heapq 路径搜索
- 多智能体环境模拟

## 主要功能

- 构建仓库地图和拣货任务环境。
- 训练智能体进行路径规划或任务选择。
- 设计多种对比算法并输出实验结果。
- 使用 Plotly/Matplotlib 生成路径和指标可视化。
- 实现 CBBA 多智能体任务分配流程。

## 工作链路

1. 构建仓库地图、货架、任务点和智能体状态。
2. 生成任务分配或拣货路径规划算例。
3. 使用强化学习或 CBBA 等算法求解任务分配和路径选择。
4. 运行对比算法并记录奖励、路径长度、完成率等指标。
5. 使用 Plotly/Matplotlib 输出路径动画、指标图和实验表格。

## 知识点

- 强化学习环境建模。
- 多智能体任务分配。
- 路径搜索和启发式算法。
- 算例实验与对比实验设计。
- 实验结果可视化。
