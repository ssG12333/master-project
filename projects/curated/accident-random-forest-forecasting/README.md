# 随机森林交通事故预测

## 项目简介

本项目围绕交通事故数量预测，完成事故数据清洗、类别特征编码、时间序列补齐、随机森林回归建模、指标评估和结果可视化。项目适合展示机器学习预测、结构化数据处理和可解释分析能力。

原始目录：`随机森林事故预测/`

## 技术栈

- Python
- pandas, NumPy
- scikit-learn
- RandomForestRegressor
- LabelEncoder
- matplotlib, seaborn

## 主要功能

- 读取交通事故记录并提取时间、天气、道路类型等特征。
- 按日聚合事故数量，补齐缺失日期。
- 对类别特征进行编码和统计分析。
- 使用随机森林回归预测未来事故数量。
- 输出 MAE、RMSE、R2 等评估指标。
- 绘制预测对比、误差分布、天气贡献和时间序列图。

## 工作链路

1. 读取事故数据，完成字段清洗和日期解析。
2. 按天聚合事故数量并补齐时间序列。
3. 对天气、道路方向、道路类型等类别变量编码。
4. 划分训练集和测试集，训练随机森林回归模型。
5. 对测试集和目标月份进行预测。
6. 输出指标、预测表和可视化图表。

## 知识点

- 结构化数据清洗和缺失值处理。
- 时间序列聚合与特征工程。
- 随机森林回归和模型评估。
- 类别特征编码。
- 预测结果可视化和误差分析。

## 关键文件

- `forecast_accidents.py`：数据处理、建模、预测和图表输出。
- `results/daily_data.csv`：按日聚合后的样例数据。
- `results/metrics.txt`：测试集指标。
- `results/actual_vs_predicted.png`：测试集预测对比。
- `results/weather_contribution.png`：天气贡献分析。
