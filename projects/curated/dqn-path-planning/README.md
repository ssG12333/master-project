# DQN 路径规划与可视化系统

## 项目简介

本项目基于 DQN 实现二维栅格环境中的路径规划，并提供 Tkinter 可视化界面、训练曲线和路径测试流程。项目适合展示强化学习环境建模、经验回放、目标网络和路径规划对比能力。

原始目录：`DQN路径规划/`

## 技术栈

- Python, PyTorch
- DQN, Replay Buffer, Target Network
- NumPy, matplotlib
- Tkinter GUI
- A* 搜索对比

## 主要功能

- 构建二维路径规划环境、障碍物和终点状态。
- 使用 DQN 智能体学习从起点到目标点的策略。
- 实现经验回放、epsilon-greedy 探索和目标网络更新。
- 提供 GUI 调参、训练、测试、保存和加载模型入口。
- 集成 A* 搜索作为路径可行性和对比参考。

## 工作链路

1. 定义栅格地图、动作空间、障碍物和终止条件。
2. 根据移动距离、碰撞、到达目标等因素设计奖励函数。
3. 使用 DQN 网络估计状态动作价值。
4. 通过 Replay Buffer 采样历史经验训练网络。
5. 周期性同步目标网络，提升训练稳定性。
6. 在 GUI 中展示路径、奖励曲线、步数变化和测试结果。

## 知识点

- DQN 算法结构和训练流程。
- 强化学习路径规划环境设计。
- 奖励函数、探索策略和目标网络。
- GUI 与训练线程交互。
- A* 与学习型路径规划的对比。

## 关键文件

- `env.py`：路径规划环境、状态转移、奖励和碰撞判断。
- `Agent.py`：DQN 网络、经验回放和智能体更新逻辑。
- `DQN.py`：训练入口、GUI、A* 对比和可视化。
