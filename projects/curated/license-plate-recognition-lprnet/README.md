# LPRNet 车牌识别系统

## 项目简介

本项目基于 LPRNet 实现车牌字符识别，重点展示轻量级卷积网络、序列识别、CTC 解码和中文车牌字符集处理能力。项目适合放在计算机视觉方向，作为目标检测之外的识别类案例。

原始目录：`LPRNet/`

## 技术栈

- Python, PyTorch
- LPRNet, CNN
- CTC Loss, Greedy Decode
- OpenCV, PIL
- NumPy, torch Dataset/DataLoader

## 主要功能

- 构建 LPRNet 车牌字符识别模型。
- 自定义车牌数据加载器和字符字典。
- 使用 CTC Loss 训练不定长字符序列。
- 支持测试集推理、贪心解码和识别结果可视化。
- 支持中文省份简称、数字、字母等车牌字符集合。

## 工作链路

1. 读取车牌图片路径，并从文件名或标注中解析车牌字符。
2. 将车牌图像缩放、归一化并转换为模型输入张量。
3. 使用 LPRNet 提取视觉特征并输出字符序列概率。
4. 通过 CTC Loss 完成训练，避免逐字符对齐标注。
5. 推理阶段使用 Greedy Decode 去重和去空白，得到最终车牌号码。

## 知识点

- 序列识别任务与 CTC 对齐机制。
- 轻量级 CNN 网络结构设计。
- 中文字符集编码和标签映射。
- 图像预处理、批处理和模型评估。
- 车牌识别端到端训练流程。

## 关键文件

- `model/LPRNet.py`：LPRNet 网络结构。
- `data/load_data.py`：车牌数据集读取和字符映射。
- `train_LPRNet.py`：训练入口。
- `test_LPRNet.py`：测试、解码和可视化入口。
