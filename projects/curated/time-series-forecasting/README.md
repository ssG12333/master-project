# 通用时间序列预测框架

## 项目简介

本项目是一个时间序列预测与实验框架，集成多种深度学习模型和数据加载模块，可用于长短期预测、异常检测、分类、插补等任务。项目重点体现时间序列模型工程、实验组织和多模型对比能力。

原始目录：`forecast/`

## 技术栈

- Python, PyTorch
- NumPy, pandas, scikit-learn
- Autoformer, Transformer, GRU, DLinear
- AGCRN, GraphPatchTST
- scipy, statsmodels, sktime
- matplotlib, tqdm

## 主要功能

- 统一数据加载和实验入口。
- 支持长周期预测、短周期预测、异常检测、分类和插补任务。
- 集成多种 Transformer 和图神经网络相关模型。
- 支持训练、测试、模型保存和结果评估。
- 提供可扩展的模型目录和实验目录结构。

## 工作链路

1. 读取 CSV 或标准时间序列数据集。
2. 构造滑动窗口、时间特征、归一化和训练/验证/测试划分。
3. 选择 Autoformer、Transformer、GRU、DLinear、AGCRN 等模型。
4. 统一训练、验证、测试和模型保存流程。
5. 输出预测结果、误差指标和对比实验结论。

## 知识点

- 时间序列窗口构造和归一化。
- Transformer 系列时间序列模型。
- 图时序模型和循环神经网络。
- 实验配置管理。
- 多模型对比实验。
