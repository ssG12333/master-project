# YOLO 密集人群检测与密度监测

## 项目简介

本项目基于 YOLOv8 实现密集人群检测，并提供 PyQt5 桌面端界面。系统支持图片和视频输入，能够检测人群目标、统计人数，并结合区域面积计算当前人群密度，适用于人流监控和安全预警场景。

原始目录：`密集人群yolo/`

## 技术栈

- Python
- YOLOv8 / Ultralytics
- PyQt5
- OpenCV
- PIL / ImageDraw
- Roboflow YOLO 格式数据集

## 主要功能

- 登录窗口和主检测界面。
- 图片检测与视频流检测。
- 检测框绘制、类别和置信度展示。
- 人数统计和历史结果记录。
- 根据输入面积计算人群密度。
- 支持中文显示字体回退。

## 工作链路

1. 使用 YOLO 格式数据集训练人群检测模型。
2. PyQt5 界面选择图片或视频输入。
3. OpenCV 读取图像帧并送入 YOLOv8 推理。
4. 对检测框进行计数、绘制和置信度展示。
5. 根据人数和区域面积计算密度，并更新界面统计结果。

## 数据与模型

数据集中包含人群检测类别，例如 `people_counterv0 - v1 2023-05-18 6-22pm`。模型权重 `best.pt` 属于大文件，后续不应直接提交到 Git，应放入 Release 或外部下载链接。
