# FIRE-YOLOv5 火灾检测与边缘推理

## 项目简介

本项目基于 YOLOv5 实现火灾目标检测，并围绕实际部署场景加入注意力模块、TensorRT 推理加速、串口通信和分块推理思路。项目适合展示目标检测模型改进和边缘设备部署能力。

原始目录：`FIRE-YOLOV5-master/`

## 技术栈

- Python, PyTorch
- YOLOv5
- OpenCV
- CBAM, SE 注意力模块
- TensorRT, CUDA
- pyserial 串口通信
- SAHI 分块推理

## 主要功能

- 火灾目标检测训练和推理。
- 在 YOLOv5 中集成 CBAM 和 SE 注意力模块。
- 使用 TensorRT 对检测模型进行推理加速。
- 将检测目标中心坐标通过串口发送给外部设备。
- 支持分块推理思路，改善大图或小目标检测效果。

## 工作链路

1. 准备火灾检测数据集并训练 YOLOv5。
2. 在模型结构中引入 CBAM/SE 注意力模块。
3. 使用 OpenCV 完成输入读取和检测结果可视化。
4. 将模型转换或加载到 TensorRT 推理流程中。
5. 将目标中心坐标编码后通过串口发送给外部设备。

## 知识点

- YOLOv5 模型结构和训练流程。
- 注意力机制在目标检测中的应用。
- TensorRT 推理部署流程。
- 串口协议和外设通信。
- 检测结果后处理、坐标转换和可视化。
