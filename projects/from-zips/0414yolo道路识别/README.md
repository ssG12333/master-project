# 基于深度学习的车道线检测系统

## 技术方向

计算机视觉、道路环境感知、驾驶辅助、车道线检测。

## 技术栈

- Python
- PyTorch
- OpenCV, NumPy, PIL
- YOLOv5 训练与推理脚本
- 视频流处理与检测结果可视化

## 工作链路

1. 读取图片或行车视频帧。
2. 对图像进行去噪、光照归一化、尺寸调整等预处理。
3. 调用深度学习模型进行车道线区域或目标检测。
4. 对检测结果进行车道线拟合、弯道判断和偏离判断。
5. 输出检测框、车道线、预警提示和可视化结果。

## 关键内容

- `code/ui.py`：检测界面和交互入口。
- `code/alarm.py`：车道偏离/弯道预警逻辑。
- `code/detect.py`, `code/predict.py`, `code/train.py`：训练、推理和预测脚本。
- `code/requirements.txt`：运行依赖。
- `code/result/`：少量检测结果截图，用于展示效果。
