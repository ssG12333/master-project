# 强化学习对抗游戏

## 技术方向

游戏 AI、强化学习环境设计、Pygame 可视化。

## 技术栈

- Python
- Pygame
- 图像资源管理
- 状态与动作逻辑设计

## 工作链路

1. 使用 Pygame 构建游戏场景和资源加载。
2. 定义角色、状态、动作和碰撞逻辑。
3. 设计对抗规则和胜负条件。
4. 运行游戏循环并渲染画面。
5. 后续可接入强化学习智能体进行策略训练。

## 关键内容

- `project.py`：游戏主逻辑。
- `clc/`：游戏图像资源。
