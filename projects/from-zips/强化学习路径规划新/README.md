# 强化学习路径规划新

## 技术方向

强化学习路径规划、算法改进、多指标对比实验。

## 技术栈

- Python
- PyTorch
- Pygame
- NumPy
- Matplotlib
- Q-learning, DQN, DDPG, A*

## 工作链路

1. 基于原版网格路径规划环境进行改进。
2. 统一状态、动作、奖励和终止条件。
3. 训练 Q-learning、DQN、DDPG 三类算法。
4. 记录奖励、损失、步数和 Q 值变化。
5. 与原版结果对比，分析改进效果。

## 关键内容

- `QLearning.py`, `DQN.py`, `DDPG.py`
- `maze.py`, `state.py`, `player.py`
- `原版/`：对照版本。
- 训练曲线图片。
