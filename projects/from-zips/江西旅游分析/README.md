# 江西旅游数据分析

## 技术方向

旅游经济分析、统计建模、计量分析、数据可视化。

## 技术栈

- Python
- pandas, NumPy
- statsmodels, linearmodels
- scipy
- Matplotlib, Seaborn

## 工作链路

1. 整理江西旅游相关指标数据。
2. 完成描述性统计和相关性分析。
3. 构建回归或面板模型分析影响因素。
4. 生成趋势图、热力图和统计表。
5. 将模型结论整理到报告中。

## 关键内容

- 分析脚本和结果图表。
- 旅游数据表和报告文档。
