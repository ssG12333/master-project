# 爬虫加深度学习预测分析

## 技术方向

数据爬虫、金融数据分析、机器学习预测、回测分析。

## 技术栈

- Python
- pandas, NumPy
- scikit-learn
- Matplotlib, Seaborn
- Excel 数据处理
- 回归模型、决策树、随机森林

## 工作链路

1. 使用爬虫脚本抓取上市公司财务数据。
2. 清洗财务指标并生成结构化 Excel 数据。
3. 对市值、每股收益、财务指标等变量做相关性和回归分析。
4. 使用线性回归、决策树、随机森林等模型预测未来指标。
5. 结合预测结果进行选股和回测分析。
6. 输出预测图、热力图、汇总表和分析报告。

## 关键内容

- `爬虫.py`：数据采集。
- `回归分析.py`：财务指标回归分析。
- `最终预测三种方法.py`：多模型预测。
- `预测的回测分析.py`：策略回测。
- `data/`：小规模结构化财务数据样例。
- `回归分析汇总/`, `最终预测三种结果/`：图表、预测结果和汇总表。
