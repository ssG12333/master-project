# 飞机路径与任务建模

## 技术方向

路径规划、数学建模、优化求解、结果可视化。

## 技术栈

- Python
- PyTorch
- NumPy
- Matplotlib

## 工作链路

1. 根据题目要求抽象飞机任务、约束和目标函数。
2. 构建路径或调度问题的数学模型。
3. 使用 Python 脚本求解不同问题分支。
4. 输出最优路径图和说明文档。
5. 将结果整理为问题说明书。

## 关键内容

- `第一问.py`, `第二问.py`：分问题求解脚本。
- `optimal_path.png`：路径可视化结果。
- `第一问说明书.docx`, `第二问说明书.docx`：建模说明。
